# Notices

EigenMiniSat is a benchmark-oriented EigenScript port target for MiniSat-style
SAT solving.

The upstream reference is MiniSat 2.2:

- Homepage: https://minisat.se/
- Source: https://github.com/niklasso/minisat
- License: MIT

MiniSat copyright notice:

MiniSat -- Copyright (c) 2003-2006, Niklas Een, Niklas Sorensson
Copyright (c) 2007-2010 Niklas Sorensson

This repository starts with an EigenScript DPLL baseline and benchmark harness.
As the port grows closer to MiniSat's core CDCL implementation, files derived
from upstream MiniSat should retain the upstream notice.
